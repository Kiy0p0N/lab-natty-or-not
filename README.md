<h1>Guia Prático: Python para Análise de Dados</h1>
<h2>📒 Descrição</h2>
<p>Este projeto consiste na criação de um eBook intitulado "Guia Prático: Python para Análise de Dados", desenvolvido utilizando técnicas de Inteligência Artificial (IA) generativa. O eBook oferece uma abordagem abrangente e fácil de entender sobre como utilizar a linguagem de programação Python para análise de dados, desde os conceitos básicos até casos de uso avançados.</p>
<h2>🤖 Tecnologias Utilizadas</h2>
<ul>
<li>ChatGPT</li>
<li>Gamma</li>
</ul>
<h2>🧐 Processo de Criação</h2>
<p>Recentemente comecei um curso de análise de dados, e isso serviu de inspiração para criar esse eBook.</p>
<p>Primeiro utilizei o ChatGPT para criar um prompt mais rico e assertivo, assim conseguindo extrair o melhor resultado possível</p>
<p>Depois, colei a saída dada pelo ChatGPT no Gamma, escolhi o formato e tema, e ele fez o resto trabalho.</p>
<p>No fim, dei alguns toques finais e revisei o texto, para garantir que estava tudo correto.</p>
<h2>🚀 Resultados</h2>
<p>O eBook apresenta um conteúdo bem estruturado e visualmente bonito, na minha opinião.</p>
